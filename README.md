# ansible-add-user
Add users and their public-key in linux(debian, redhat) remote hosts
# Requirements
None.
# Role Variables
see `vars/users.yml`
# Dependencies
None.
# Example Playbook
```
- host: all
  role:
   - ansible-add-user
```
